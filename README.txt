Projet Parkease:
Ce projet est un simulateur de parking avec une base de données. Ce programme prend en compte les entrées/sorties et donne le tarif à la sortie pour 2.40/h


Pré-requis:
- Python 3


Fonctionnalités:
- Enregistrement des plaques d'immatriculation entrant dans une base de données.
- Calcul du coût du stationnement en fonction de la durée.
- Limite de place dans le parking limitée à 25.


Arborescences et but des fichiers:

parkease/
│── parkease.py - Programme principal exécutant toutes les fonctions.
│── parking.py - Gestion des entrées et sorties.
│── database.py - Gestion de la base de données.
│── config.py - Fichier de configuration ( tarif et des places totales ).
├── data/
│   ├── bdd.json - Base de données contenant l'heure d'entrée et la plaque d'immatriculation de la voiture.
